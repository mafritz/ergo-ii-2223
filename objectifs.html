Objectifs du cours ERGO II

Le cours permet de mettre en pratique des principes et techniques liés à l'UX, terme anglais pour designer l'expérience utilisateur, dans la conception et évaluation d'interfaces web. Il applique les connaissances théoriques et techniques abordées dans le cours ERGO I.

À la fin du cours, vous aurez acquis les connaissances et compétences suivantes :

  • Connaître les différentes étapes dans la gestion d'un projet UX ;
  • Produire des prototypes à différents degrés d'abstraction, du prototype papier à une maquette interactive fonctionnelle ;
  • Intégrer des principes comportementaux, productifs et de design dans la conception et évaluation d'une interface utilisateur ;
  • Appliquer des techniques d'architecture de l'information ;
  • Mener un test utilisateur avec analyse d'indices de performance et de perception d'une interface utilisateur ;
  • Apporter et justifier des remédiations sur la base d'informations issues d'un test ou de principes établis dans la littérature UX.

Productions

Pour atteindre et consolider les objectifs du cours, vous allez produire les éléments suivants :

  • Deux versions d'une maquette interactive d'un site web, dont la deuxième est améliorée suite aux résultats d'un test utilisateur conçu et mené par vos soins
  • Un rapport de conception et évaluation d'un site web destiné aux parties prenantes dans un projet fictif ou appliqué
  • Un prototype papier ou une maquette interactive d'un site ou application web pour un dispositif mobile
  • Un rapport incluant les résultats quantitatifs d'un test A/B fictif qui compare deux versions d'une interface web
