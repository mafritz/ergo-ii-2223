Support

Liste de ressources utiles pour le cours.

Slides

Supports utilisés pendant les séances du cours synchrone. Attention : les liens pointent vers des slides en HTML5 qui peuvent évoluer dans le temps. Si vous désirez la version des slides utilisées pendant le cours, vous pouvez les télécharger depuis le dépôt GitHub.

UX pour la conception et évaluation de UI pour le web
Présentation général du domaine UX/UI. La présentation se focalise principalement sur la phase d'implémentation d'une interface concrète. Les techniques de conception et évaluation des phases exploratoires (e.g. analyse des besoins, persona, ...) ne sont pas abordées.
Prototypage low-fidelity dans l'UX
Présentation sur des techniques de prototypage low-fi dans le UX. Ce type de prototypage s'effectue surtout dans des phases initiales du projet, afin de maximiser l'exploration de variantes en minimisant le travail à accomplir au niveau du design et des fonctionnalités.
Prototypage high-fidelity dans l'UX
Présentation sur des techniques de prototypage high-fi dans le UX. Ce type de prototypage essaie de véhiculer une expérience la plus proche du produit finale au niveau de l'apparence esthétique et/ou des fonctionnalités.
Techniques d'évaluation dans l'UX
Présentation sur des techniques d'évaluation comme l'oculométrie, le test A/B, le test des 5 seconds, les échelles d'utilisabilité/UX, et le test utilisateur.

Applications de prototypage

Applications pour le prototypage à différents niveaux d'abstraction.

Figma
Application pour le prototypage et les maquettes interactives qui permet de collaborer en temps réel. L'application web marche dans le navigateur ou peut être téléchargée en version desktop. Le plan gratuit permet une collaboration limitée.
Uizard
Application collaborative pour le prototypage rapide. L'application permet d'importer des prototypes papier et les transforme directement en dessin.
Balsamiq
Application pour le prototypage surtout low-fidelity. L'application peut être utilisée dans le cloud ou en modalité application desktop.
Adobe XD
Application de prototypage de la suite Adobe, il s'intègre avec les autres applications de la suite (e.g. Illustrator, Photoshop, etc.).
Justinmind
Application de prototypage pour différents dispositifs. Possibilité de collaborer avec d'autres personnes.
Invision
Plateforme pour la production de produits digitaux. Possibilité de collaborer avec d'autres personnes.

Autre Applications

Autres applications souvent utilisés dans le domaine UX.

Miro
Application web qui permet de créer des tableaux blancs collaboratifs. L'application fonctionne dans le navigateur ou peut être téléchargée comme application desktop.
Trello
Application pour la gestion de projets. Elle permet par exemple de créer des Kanban boards souvent utilisés dans le design/développement.

Sites

Sites avec informations utiles au domaine UX.

UX Myths
Collection de mythes sur le domaine UX disponibles dans plusieurs langues, y compris le français. La liste date déjà de quelques années, mais elle est toujours actuelle.
Interaction Design Foundation
Site qui propose une section Literature avec des fiches et articles approfondies sur la thématique de l'UX et de l'interaction personne-machine plus en général.
A List Apart
L'un des premiers sites à propos de la création de sites web, avec des articles du type UX, mais également plus techniques.
All About UX
Collection de ressources pour les professionnels du UX. Le site n'est probablement plus mis à jour régulièrement, mais propose entre autre une liste intéressante de définition du domaine UX.
W3C Accessibility Standards Overview
Guidelines du W3C sur la conception et l'évaluation de l'accessibilité des interfaces web.
